---
title: Get Creator
description: Retrieve a creator profile by ID.
---

## GET /v1/creators/:id

### Example Request

```bash
curl https://api.boobie.ai/v1/creators/cr_abc123 \
  -H "Authorization: Bearer YOUR_BOOBIE_API_KEY"
```

### Response

```json
{
  "id": "cr_abc123",
  "name": "Creator Name",
  "email": "creator@example.com",
  "vertical": "music",
  "status": "active",
  "ip_count": 14,
  "created_at": "2026-06-12T09:00:00Z"
}
```

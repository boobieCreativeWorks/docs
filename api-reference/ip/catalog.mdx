---
title: IP Catalog
description: Query the IP catalog for a creator or organization.
---

## GET /v1/ip/catalog

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `creator_id` | string | Filter by creator |
| `type` | string | Filter by IP type |
| `limit` | integer | Results per page (max 100) |
| `offset` | integer | Pagination offset |

### Response

```json
{
  "items": [
    {
      "id": "ip_abc123",
      "title": "Song Title",
      "type": "song",
      "timestamp": "2026-06-12T09:00:00Z",
      "status": "registered"
    }
  ],
  "total": 14,
  "limit": 20,
  "offset": 0
}
```
